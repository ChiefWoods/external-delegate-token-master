# External Delegate Token Master

[Source Repository](https://github.com/ChiefWoods/external-delegate-token-master)

## Getting Started

### Prerequisites

1. Update your Solana CLI

```sh
agave-install update
```

### Setup

1. Clone the repository

```sh
git clone https://github.com/ChiefWoods/external-delegate-token-master.git
```

2. Resync your program id

```sh
quasar keys sync
```

4. Build the program

```sh
quasar build
```

#### Testing

Run all tests.

```sh
quasar test
```
